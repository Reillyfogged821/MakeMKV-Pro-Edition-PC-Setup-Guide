## ⚠️ Solving Common Errors
Users often see errors during the setup process. Use these fixes if you encounter problems.

### Missing Drive Support
If the software does not see your disc drive, update your Windows drivers. Open "Device Manager," find your DVD/Blu-ray drive, right-click it, and select "Update driver."

### Setup Failed
Sometimes the installer stops before it finishes. This happens if an old version of the software remains on your system. Go to your Windows Settings, select "Installed Apps," and remove any previous versions of MakeMKV before you start the new installation.

### Permission Denied
If the app tells you it lacks permission to write files, create a new folder on your Desktop and point the software to save files there instead of the default "Program Files" directory.
